Internet Query Engine %version

Welcome to IQE! This page gives you the lowdown on how to tweak and tune your server with some nifty command-line options.

Command-Line Options

Option Short Form Description Current Value
--port -p Port number to run the server on (0-65535) %port
--backlog -b Max pending connections (0 = system default) %backlog
--threads -t Number of threads in the pool (0 = synchronous) %threads
--debug -d Enable debug mode (true/false) %debug

Try running the server with different options, like java -jar iqe-%version.jar -p 8080 -t 10 -d, and watch this page update!
